# SchoolDoc

Интеллектуальный генератор школьных заявлений и записок, разработанный на базе PyQt6. Программа помогает родителям мгновенно создавать корректные документы с автоматической подстановкой данных.

---

###  Основные функции

- **Автоматизация шаблонов**: Быстрая генерация документов («Записка об отсутствии», «Уход из школы самостоятельно», «Освобождение от физкультуры»).
- **Умный текстовый редактор**: Возможность ручной правки сгенерированного текста перед сохранением.
- **Интеллектуальный экспорт**: Сохранение в формат `.txt` с автоматическим выравниванием строк (макс. 70 символов) и контролем переноса слов.
- **Персонализация**: Система настроек сохраняет данные (ФИО родителя, ребенка, класс) в локальный файл `settings.data`, избавляя от повторного ввода.
- **Валидация**: Автоматическая подстановка текущей даты и проверка корректности введенных данных.

###  Технологический стек

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![Qt](https://img.shields.io/badge/Qt-217346?style=for-the-badge&logo=Qt&logoColor=white)

- **UI Framework**: PyQt6 (использование сигналов, слотов и многооконного интерфейса).
- **Standard Libraries**: `datetime` (работа с датами), `sys`, `traceback`.
- **Data Persistence**: Хранение настроек в структурированном текстовом формате.

---

###  Структура проекта

- `Document.py` — главный файл приложения, содержащий логику основного окна (`StatementApp`) и окна настроек (`InputWindow`).
- `ТЗ.docx` — подробная проектная документация с описанием алгоритмов.
- `settings.data` — (генерируется автоматически) файл для хранения пользовательских данных.

###  Инструкция по запуску

Для работы программы требуется установленная библиотека PyQt6.

1. **Установите зависимости:**
   ```bash
   pip install PyQt6
2. **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/kepchupchik/school-docs-helper.git](https://github.com/kepchupchik/school-docs-helper.git)
    cd school-docs-helper
3. **Запустите программу:**
``` bash
   python Document.py


###  **Как пользоваться:**
Откройте вкладку "Настройки" и один раз введите ФИО родителя, ребенка и класс.

Выберите нужный тип заявления из выпадающего списка.

Проверьте текст в редакторе и нажмите "Сохранить документ".

Файл будет готов к печати в формате .txt.
